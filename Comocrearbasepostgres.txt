Como configurar una base de datos en postgres:
1. Crear estructura de datos
initdb -D <data>
--2. Iniciar el servicio de la base de datos 
 pg_ctl -D <data> start. 
3. Registrar el servicio, como servicio de windows
pg_ctl register -N postgres -D data 
4. Conectarse con la base de datos en la consola
psql -U Delia -d postgres
5 Crear tabla ejemplo
 create table colores (
nombre varchar (24), 
color varchar (24)
);
6. selecccionar los datos
select <>* from colores;
